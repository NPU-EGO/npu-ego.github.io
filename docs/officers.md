以下為社團幹部架構，若有調整請直接更新本頁：

| 職務 | 主要職責 | 聯絡方式 |
| --- | --- | --- |
| 社長 | 對內對外代表，主持社務會議，統籌活動與課程 | `president@example.edu.tw` |
| 副社長 | 協助社長，社長不在時代理 | `vp@example.edu.tw` |
| 總務 | 財務收支、會費與經費管理，定期公開報表 | `finance@example.edu.tw` |
| 活動長 | 活動規劃與執行，場地與資源協調 | `events@example.edu.tw` |
| 教學組組長 (Training Leader) | 課程規劃、教材與講師協調，學習進度追蹤 | `training@example.edu.tw` |
| 技術組組長 (Tech Leader) | 攻防實驗環境維護、工具選型與技術支援 | `tech@example.edu.tw` |
| 公關 | 對外聯繫、社群經營、宣傳合作 | `pr@example.edu.tw` |
| 美宣 | 視覺設計、品牌一致性、素材製作 | `design@example.edu.tw` |

**指導老師**：

- 姓名：_待補_  
- 聯絡：`advisor@example.edu.tw`  
- 角色：提供專業指導、校內行政協調、活動與課程建議。
