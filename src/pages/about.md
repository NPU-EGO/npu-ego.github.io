# 關於我們

## 終端防護攻防隊 (EndPoint Guardian Ops)

EGO 成立於 2025 年，由一群熱愛資訊安全的學生所創立。我們的目標是打造一個能夠自由交流、切磋技術的平台。

### 我們的使命
1.  **推廣資安意識**：讓更多人了解資訊安全的重要性。
2.  **技術深耕**：提供紮實的技術訓練，從 Web Security 到 Binary Exploitation。
3.  **實戰應用**：透過 CTF 競賽與靶機演練，將理論轉化為實戰能力。

### 社團特色
*   **Mentor 制度**：學長姐手把手教學。
*   **資源豐富**：擁有專屬實驗環境與資源。
*   **與業界接軌**：透過澎湖資安健檢推廣與業界合作，進行安全評估與技術交流，達到學以致用。
*   **在地服務**：推廣澎湖地區資安意識，協助在地企業與組織提升資安防護能力。

### 加入我們
無論你是資安大神還是剛入門的新手，只要有熱忱，EGO 都歡迎你的加入！
